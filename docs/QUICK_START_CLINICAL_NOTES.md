### Scenario 2: Patient with RDS (Respiratory Distress)

**Time Required:** **1-2 minutes** ‚ö°

#### Steps:
1. Click **"Add New Note"**
2. Fill in **Vital Signs** (temperature, HR, RR, SpO2, etc.)
3. Click **"RDS Pattern"** quick template button
4. Review **CHEST** section - pre-selected with:
   - ‚úì Subcostal retractions
   - ‚úì Intercostal retractions
   - ‚úì Grunting present
   - ‚úì Nasal flaring
5. Adjust if needed (add/remove findings)
6. In **Custom Findings** box, add: "On CPAP 6 cm H2O, FiO2 40%"
7. Add **Medications** (Caffeine, Antibiotics, etc.)
8. Write **Assessment/Plan** in Clinical Note section
9. Click **"Save Progress Note"**

**Result:** Complete, professional RDS documentation in under 2 minutes!

### Scenario 4: Custom Examination (No Template)

**Time Required:** **2-3 minutes** (still faster than typing!)

#### Steps:
1. Click **"Add New Note"**
2. **DON'T** click any template
3. Go through each system **one by one**:

#### **CNS Section:**
- Expand the CNS accordion
- See **Normal Findings** (green section) and **Abnormal Findings** (red section)
- Click checkboxes for relevant findings:
  - ‚úÖ Alert and active
  - ‚úÖ Good cry, strong
  - ‚ùå (Don't select Lethargy, Seizures, etc. if not present)
- Add custom text: "Anterior fontanelle 2x2 cm, soft and flat"
- See real-time **Preview** at bottom

#### **CVS Section:**
- Select normal findings:
  - ‚úÖ S1S2 heard, regular
  - ‚úÖ CRT < 2 seconds
- OR select abnormal:
  - ‚ö†Ô∏è Murmur detected (systolic/diastolic)
- Add custom: "Grade 2/6 systolic murmur at left sternal border"

#### **CHEST Section:**
- Select findings as needed
- Add custom observations

#### **PER ABDOMEN Section:**
- Select findings
- Add custom notes

4. Click **"Save Progress Note"**

## üéØ Pro Tips

### Tip 1: Use "Copy Last" for Stable Patients
If vitals haven't changed much:
1. Click **"Copy Last"** button (purple button at top)
2. Vitals and medications copied from previous note
3. Adjust examination findings
4. Save!

### Tip 2: Clear Individual Systems
Made a mistake? Click the **"Clear"** button next to each system header to reset just that system.

### Tip 3: Use Custom Fields Liberally
The checkbox options are comprehensive, but **custom fields** let you add:
- Specific measurements
- Time-based observations
- Rare findings
- Patient-specific notes

Example:
```
Custom Finding: "Oxygen requirement decreased from 40% to 30% over last 6 hours"
```

### Tip 4: Preview Before Saving
Each system shows a **preview box** at the bottom showing exactly how your selections will appear in the final note.

### Tip 5: Combination Selection
You can select BOTH normal AND abnormal findings if needed:
- ‚úÖ Bilateral equal air entry (normal)
- ‚ö†Ô∏è Grunting present (abnormal)

This gives you complete flexibility!
